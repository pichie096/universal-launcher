## Installation

### 1. Install Python
When installing Python:

- ❌ Uncheck **“Use admin privileges”**
- ✅ Check **“Add Python to PATH”**

---



### 2. Install Required Libraries
Open **Command Prompt** and run:


pip install minecraft-launcher-lib requests psutil
